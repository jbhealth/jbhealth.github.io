# jbhealth.github.io

The JB Health landing page — *built for bodies with mileage*.

Served via GitHub Pages at **[jbhealth.co.uk](https://jbhealth.co.uk)**.

## Files

| File | Purpose |
|------|---------|
| `index.html` | The landing page (single, self-contained file) |
| `404.html` | Branded "page not found" fallback |
| `CNAME` | Custom-domain binding to `jbhealth.co.uk` (created by GitHub when the custom domain is set in Settings → Pages) |

## Editing

The page is one standalone HTML file with all CSS inline — no build step, no dependencies. Edit `index.html` directly (here on GitHub or locally), commit to `main`, and the live site updates within a minute or two.

The intake form is hosted on Tally; the **Apply** buttons link out to it (three places: navbar, hero, final CTA).
